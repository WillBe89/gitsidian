# Roadmap

Ideas for future releases — roughly in priority order, not firm commitments.
Suggestions and contributions welcome; sponsorship helps fund the signing/notarization
that removes the OS security prompts (see the README's **Support** section).

## Polish & safety
- **Confirm before closing a running tab** — guard against accidentally closing a
  session that's mid-task (e.g. an agent still working).
- **Settings / personalization** — theme & accent colour, light/dark, font size,
  terminal font, default AI, scrollback length.
- **Keyboard shortcuts** — ⌘T new terminal, ⌘W close tab, ⌘1–9 switch tabs, ⌘K clear.

## Git & GitHub
- **Pull preview** — before pulling, show the incoming files and approximate
  download size, so limited-storage users can decide.
- **Mark as ignore** — right-click a file/folder → add to `.gitignore`.
- **Inline diff** — click a changed file to see exactly what changed.
- **AI-assisted commit messages** and a quick branch switcher.

## Sessions
- **Session persistence** — restore open tabs/projects on relaunch.
- **Split terminals** — two sessions side by side in one tab.
- **"Agent finished" notifications** — a system notification when a background tab
  goes from busy → idle.
- **Paste an image → file** for AI sessions (great for screenshots).

## Distribution
- **Code signing & notarization** (macOS + Windows) to remove the security warnings.
- **Auto-update** so new versions install without a manual re-download.
- **Linux packaging** (AppImage / `.deb`).
